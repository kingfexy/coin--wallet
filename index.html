WALLET CONNECT

Connecting and Protecting Wallets

What is WalletConnect?

WalletConnect is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any app from their mobile phone, making WalletConnect wallets a safer choice compared to desktop or browser extension wallets.

How does it work?

WalletConnect connects web applications to supported mobile wallets. WalletConnect session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).
